SentinelHub Analysis Tool
Satellite Image Processing & Environmental Index Generation Tool.
Project Details
This Python-based application enables users to dynamically define bounding boxes (areas of interest) and extract satellite imagery using the SentinelHub API. Designed for environmental monitoring and geospatial analysis, the tool generates key vegetation and water indices, offering valuable insights for agriculture, land use, and ecological research.
Tools & Technologies: Python, SentinelHub API, NumPy, Rasterio, Docker, GeoJSON, NDVI/NDWI calculation
