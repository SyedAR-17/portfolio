Revolutionizing EV Charging Infrastructure
An Integrated IoT, GIS, and Mobility-Based Solution for Urban E-Mobility
Project Details
This multi-phase project aimed to transform the EV charging ecosystem by developing a scalable, data-driven solution tailored for electric two-wheelers. Combining IoT, mobile technology, and geospatial analysis, the project tackled the key challenges of battery management, user accessibility, and infrastructure planning.
To ensure efficient and accessible EV charging infrastructure, a site suitability analysis was conducted using Geographic Information System (GIS) techniques. The analysis focused on identifying optimal locations for EV charging stations in Islamabad by evaluating multiple spatial and non-spatial factors. Key criteria included proximity to major roads, population density, existing power grid infrastructure, commercial activity zones, and public transport hubs. Each factor was weighted using the Analytical Hierarchy Process (AHP) to reflect its importance in supporting EV adoption.
